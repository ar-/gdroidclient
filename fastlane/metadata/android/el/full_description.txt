# Τι είναι αυτή η εφαρμογή

Με το G-Droid μπορείτε να περιηγηθείτε στο ελεύθερο αποθετήριο εφαρμογών F-Droid. Είναι εναλλακτικό της επίσημης εφαρμογής.

# Επιπλέον δυνατότητες από την επίσημη εφαρμογή

* Δείχνει κριτικές και σχόλια για τις εφαρμογές
* Βαθμολογία (έως 5 αστέρια)
* Δείχνει βαθμολογία από άλλες πλατφόρμες (github, gitlab, sourceforge)
* Περισσότερα στοιχεία (όπως μέσος χρόνος ανάμεσα στις εκδόσεις), και βαθμολογία σύμφωνα με αυτά
* Οι χρήστες επιλέγουν ποια στοιχεία είναι σημαντικά για τις βαθμολογίς των εφαρμογών, έτσι δεν υπάρχει "η καλύτερη εφαρμογή" αλλά "η καλύτερη εφαρμογή για τον καθένα"
* Δείχνει "παρόμοιες εφαρμογές" 'η "εφαρμόγες από την ίδια κατηγορία" για κάθε εφαρμογή που βλέπετε
* Δείχνει "εφαρμογές από τον ίδιο προγραμματιστή" για κάθε εφαρμογή που βλέπετε
* Δείχνει λίστα με προγραμματιστές, με τις εφαρμογές τους, και τη μέση βαθμολογία
* Δείχνει την ιστοσελιίδα της εφαρμογής - έστω κι αν αυτή είναι απλά ένα README αρχείο στο GitHub. Το F-Droid πρόσφατα αφαίρεσε πολλές ιστοσελίδες από τις περιγραφές γιατί δεν ήταν σε λειτουργική ιστοσελίδα
* Δείχνει τις νέες εφαρμογές και τις πρόσφατα αναβαθμισμένες ξεχωριστά
* Δείχνει τις καρτέλες των εφαρμογών στο ίδιο μέγεθος για καλύτερη προβολή
* Παρέχει προβολή σε λίστα (όπως οι παλαιότερες εκδόσεις της εφαρμογής F-Droid)
* Περισσότερες επιλογές κατάταξης των εφαρμογών 
* Επιλογή για αγαπημένες εφαρμογές 
* Επιλογή για απόκρυψη εφαρμογών
* Δείχνει τις κατηγορίες με σωστές μεταφράσεις 
* Δείχνει τα screenshots των εφαρμογών στη σωστή γλώσσα
* Καλύτερο για μεγάλες οθόνες, δείχνει πιο πολλές εφαρμογές στα tablet
* Tag για να βρίσκετε εφαρμογές για συγκεκριμένες εργασίες, καθώς οι κατηγορίες του F-Droid είναι πολύ γενικές
* Προβολή "τυχαίων εφαρμογών" για να βρίσκετε εφαρμογές που αλλιώς δε θα βρίσκατε
* Δείχνει εφαρμογές που μπορεί να αρέσουν στο χρήστη (χωρίς να χρησιμοποιεί τα δεδομένα του για ανάλυση)
* "Διαμοιρασμός" εφαρμογών έτσι ώστε να μπορούν να ανοίγουν και με άλλες εφαρμογές (F-Droid, Yalp, Aurora, ...), ώστε να βλέπετε περισσότερες πληροφορίες για την εφαρμογή
* Διαφορετικός αλγόριθμος εύρεσης, ώστε να γίνεται εύρεση πιο συγκεκριμένα και αποτελεσματικά
* Δείχνει τα εικονίδια των εφαρμογών (εάν η σύνδεση στο ίντερνετ είναι καλή)
* Δείχνει περισσότερα screenshots
* Έχει μοντέρνο και νέο κώδικα πλήρως ανανεωμένο
*  To G-Droid είναι υπό συνεχή εξέλιξη (Η εφαρμογή του F-Droid δεν εξελίσσεται πλέον)
* Πολλά χαρακτηριστικά θα προστεθούν ακόμη (Το G-Droid είναι απλά λίγων εβδομάδων) ...

# Πράγματα που το G-Droid δεν μπορεί να κάνει (ακόμα) σε σχέση με το F-Droid 

* Να δείξει ή εγκαταστήσει παλαιότερες εκδόσεις των εφαρμογών 
* Να δημιουργήσει τοπικό αποθετήριο, να μοιραστεί εφαρμογές με τρίτους μέσω bluetooth 
* Να προσθέσει αλλά αποθετήρια εκτός του F-Droid 
* Να ενημερώνει τις εφαρμογές αυτόματα 
*Να έχει ρυθμίσεις για htyp proxy
* Το G-Droid σας βοηθά με διάφορους τρόπους να βρείτε καλύτερες εφαρμογές, αλλά κάποιες φορές μπορεί να χρειαστείτε το κουμπί "διαμοιραμός" για να ανοίξετε το F-Droid για την εγκατάσταση. Το G-droid δεν είναι ακόμα πλήρης αντικαταστάτης της εφαρμογής του F-Droid

# Χαρακτηριστικά που είναι ίδια στο G-Droid και στο F-Droid

* Δείχνει τις ίδιες εφαρμογές, το G-Droid δεν προσθέτει, αλλάζει τις εφαρμογές ή τα λινκ 
*Όλα τα μεταδεδομένα παρέχονται από το  F-Droid (π.χ. το link για το github)
* Είναι άδεια GPL3

# Εξήγηση των αδειών

* INTERNET: για να κατεβαίνουν οι εφαρμογές
* WRITE_EXTERNAL_STORAGE: Για να αποθηκεύονται προσωρινά τα αρχεία εγκατάστασης των εφαρμογών (υποχρεωτικό για παλιές εκδόσεις Android)
