# Τι είναι αυτή η εφαρμογή

Με το G-Droid μπορείτε να περιηγηθείτε στο αποθετήριο ελεύθερων εφαρμογών F-Droid. Είναι εναλλακτικό της επίσημης εφαρμογής.

# Πρόσθετα χαρακτηριστικά σε σχέση με το F-Droid

* Εμφανίζει κριτικές και σχόλια
* Αξιολογήσεις εφαρμογών (έως 5 αστέρια) - οι μετρήσεις δημιουργούνται μηχανικά
* Εμφανίζει αξιολογήσεις από άλλες πλατφόρμες (github, gitlab, sourceforge)
* Περισσότερα στοιχεία (π.χ. μέσος χρόνος ανάμεσα στις εκδόσεις), και αξιολόγηση σύμφωνα με αυτά
* Οι χρήστες επιλέγουν ποιες μετρήσεις είναι σημαντικές για αυτούς, δεν υπάρχει η καλύτερη εφαρμογή - υπάρχει μόνο η καλύτερη για τον τρέχοντα χρήστη
* Εμφανίζει παρόμοιες και της ίδιας κατηγορίας εφαρμογές όταν βλέπετε μία εφαρμογή
* Εμφανίζει εφαρμογές του ίδιου προγραμματιστή όταν βλέπετε μια εφαρμογή
* Εμφανίζει λίστα με όλους τους προγραμματιστές, τις εφαρμογές τους και τη μέση βαθμολογία σε αστέρια
* Εμφανίζει τον ιστότοπο της εφαρμογής, ακόμα κι αν είναι ένα αρχείο README
* Εμφανίζει τις νεότερες και τις πρόσφατα ενημερωμένες ξεχωριστά
* Εμφανίζει τις καρτέλες εφαρμογών στο ίδιο μέγεθος για καλύτερη επισκόπηση
* Παρέχει μια προβολή λίστας (όπως σε παλαιότερες εκδόσεις F-Droid) για όσους δεν τους αρέσουν οι καρτέλες
* Περισσότερες επιλογές ταξινόμησης
* Δυνατότητα επισήμανσης εφαρμογών με «αστέρι»
* Δυνατότητα απόκρυψης εφαρμογών και αφαίρεσή τους από όλες τις προβολές
* Εμφανίζει τις κατηγορίες σε σωστή μετάφραση (αν υπάρχει), μετά από αλλαγή γλώσσας συσκευής
* Εμφανίζει στιγμιότυπα οθόνης στη σωστή γλώσσα
* Καλύτερη χρήση μεγαλύτερων οθονών: Περισσότερες εφαρμογές εμφανίζονται σε tablet
* Ετικέτες για ευκολότερη εύρεση εφαρμογών (πελάτες ηλεκτρονικού ταχυδρομείου, προγράμματα περιήγησης αρχείων, επεξεργαστές κειμένου), καθώς οι κατηγορίες του F-Droid είναι πολύ γενικές
* Προβολή τυχαίων εφαρμογών για να ανακαλύψετε εφαρμογές
* Εμφανίζει εφαρμογές που μπορεί να αρέσουν στον χρήστη (χωρίς να κλέβει τα δεδομένα του)
* Στοχεύει στην παροχή διαφορετικών προβολών του αποθετηρίου, ώστε να βοηθήσει το χρήστη στην εύρεση κατάλληλων εφαρμογών και να ενθαρρύνει τη χρήση περισσότερων εφαρμογών από το αποθετήριο
* «Κοινή χρήση» εφαρμογών, ώστε να μπορούν να ανοίγουν με άλλα καταστήματα (F-Droid, Yalp, Aurora, …), για να λαμβάνετε περισσότερες πληροφορίες.
* Διαφορετικός αλγόριθμος αναζήτησης, μπορεί να αναζητήσει πιο εντατικά και να συνεχίσει μια αναζήτηση αν τα αποτελέσματα είναι ανεπαρκή
* Εμφανίζει τα εικονίδια εφαρμογών (αν η σύνδεση είναι καλή)
* Εμφανίζει περισσότερα στιγμιότυπα οθόνης
* Εμφανίζει στιγμιότυπα οθόνης, πλήρως ενημερωμένα και στη σωστή γλώσσα
* Εμφανίζει στιγμιότυπα οθόνης, ακόμα και όταν δεν βρίσκονται στη δομή «fastlane»
* Νέο, σύγχρονο εύκολα διατηρήσιμο κώδικα
* Ενεργά διατηρημένο
* Πολλά ακόμα θα έρθουν

# Πράγματα που το G-Droid δεν κάνει (ακόμα) σε σχέση με το F-Droid

* Να εμφανίσει ή εγκαταστήσει μια παλαιότερη έκδοση
* Να δημιουργήσει τοπικό αποθετήριο, να μοιραστεί εφαρμογές με τρίτους «κοντά»
* Να προσθέσει αποθετήρια εκτός του F-Droid
* Να χρησιμοποιήσει την επέκταση με δικαιώματα (ολοκληρώθηκε, αλλά δεν έχει ρυθμιστεί), ωστόσο το G-Droid μπορεί να χρησιμοποιήσει δικαιώματα υπερχρήστη (root) σε rooted συσκευές για εύκολες εγκαταστάσεις
* Αυτόματες ενημερώσεις
* Ρυθμίσεις διακομιστή μεσολάβησης HTTP
* Το G-Droid βοηθά με διάφορους τρόπους να βρείτε καλύτερες εφαρμογές, ίσως χρειαστείτε ακόμα το κουμπί "κοινή χρήση" για να ανοίξετε την εφαρμογή στο F-Droid και να την εγκαταστήσετε. Το G-Droid αυτή τη στιγμή δεν αποτελεί ακόμη πλήρη αντικατάσταση της εφαρμογής F-Droid.

# Κοινά χαρακτηριστικά του G-Droid και του F-Droid

* Εμφανίζει τις ίδιες εφαρμογές - Το G-Droid δεν προσθέτει εφαρμογές ούτε τροποποιεί τις εκδόσεις, τις υπογραφές ή τις διευθύνσεις λήψης
* Όλα τα μεταδεδομένα παρέχονται από το F-Droid (π.χ. ο σύνδεσμος προς το GitHub)
* Άδεια GPLv3+

# Επεξήγηση αδειών

* INTERNET: Για λήψη εφαρμογών και μεταδεδομένων
* WRITE_EXTERNAL_STORAGE: Για προσωρινή αποθήκευση των εφαρμογών (μόνο σε παλαιότερες εκδόσεις Android)
